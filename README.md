# Cosmos Medical Technologies Website

Enterprise-grade landing page for Cosmos Medical Technologies, built with Next.js 15, Tailwind CSS, and Framer Motion.

## Setup Instructions

1. Ensure you have Node.js installed.
2. Unzip this project directory.
3. Open a terminal in the project root (`cosmos-med-tech`).
4. Run `npm install` to install all dependencies.
5. Run `npm run dev` to start the development server.
6. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for seamless deployment on Vercel:
1. Push to GitHub.
2. Import project in Vercel.
3. Deploy!
